# 旬食工房きよし ウェブサイト

宮崎県東臼杵郡門川町にある旬食工房きよしのウェブサイトです。1980年創業、46年の歴史あるチキン南蛮が看板メニューの飲食店です。

## プロジェクト概要

- **場所**: 宮崎県東臼杵郡門川町
- **創業**: 1980年（46年の歴史）
- **看板メニュー**: チキン南蛮
- **形態**: 定番・家庭料理
- **ターゲット層**: 地元の住民、観光客、ファミリー層

## 専門家の視点

このウェブサイトは以下の4つの専門家の視点から作成されました：

1. **WEBデザイナー**: 和風・伝統的なデザイン、レスポンシブ対応、直感的なナビゲーション
2. **コピーライター**: 「門川町で46年、変わらぬ味を」をコンセプトにしたストーリーテリング
3. **QAテスター**: アクセシビリティ、モバイル対応、パフォーマンス最適化

## カスタマイズ

### 画像の追加
`images/` フォルダに実際の写真を追加してください。ファイル名は `index.html` と対応させてください。

### 連絡先の更新
`index.html` 内の以下の部分を実際の連絡先に更新してください：

- 電話番号: `0982-XX-XXXX`
- Instagramリンク: `https://www.instagram.com/` の後にアカウント名を追加
- Google Mapsリンク: `宮崎県東臼杵郡門川町+旬食工房きよし`

### 色の変更
`styles.css` 内の `:root` セクションでカスタマイズできます：

```css
:root {
    --primary: #8B4513;    /* 金茶 */
    --secondary: #2F4F4F;   /* 墨汁グレー */
    --accent: #C0392B;      /* 朱色 */
    /* ... その他の色設定 */
}
```
