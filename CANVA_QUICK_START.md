# 🎨 Canvaメニュー表 作成クイックスタート

## 📝 準備するもの
- [ ] Canvaアカウント（無料でOK）
- [ ] 上記のメニューテキストファイル（[CANVA_MENU_TEXT.txt]）
- [ ] 15〜20分の時間

---

## 🚀 作成手順（ステップバイステップ）

### ステップ1: Canvaにアクセス（1分）
1. [Canva](https://www.canva.com) にアクセス
2. ログインまたは新規登録（無料）
3. 右上の「デザインを作成」をクリック
4. 「カスタムサイズ」をクリック
5. 幅: `210`、高さ: `297`、単位: `mm` を入力
6. 「新しいデザインを作成」をクリック

### ステップ2: 背景を設定（2分）
1. 左側のメニューから「背景」をクリック
2. カラーコード入力欄に `#fff9e6` と入力
3. 背景がクリーム色になります

**※ もっと和風にしたい場合**:
- 「要素」→「アップロード」→「メディアをアップロード」
- 和風の背景素材がある場合はアップロード

### ステップ3: ヘッダーを作成（3分）
1. 「テキスト」→「見出しを追加」をクリック
2. 以下のテキストを入力：

```
旬食工房きよし
Shunshoku Kobo Kiyoshi
〜 宮崎県門川町 〜
46年の想い、ふるさとの味
```

3. フォントを変更:
   - 「旬食工房きよし」: 60pt、太字、色 #8b4513
   - 英文: 24pt、色 #8b4513
   - 副タイトル: 18pt、色 #d4af37

4. 中央揃えに配置

### ステップ4: 看板メニューを追加（5分）
1. 「テキスト」→「見出しを追加」
2. 以下をコピー＆ペースト:

```
★★★ 看板メニュー ★★★

チキン南蛮
Chicken Nanban

創業当時の味をベースに、日々改良を重ねてきた私たちの誇るチキン南蛮です。ジューシーな鶏肉に、特製タルタルソースを合わせました。

¥1,200（税込）
```

3. 装飾を追加:
   - 「要素」→「線」で上下に線を引く
   - 線の色: #d4af37（金色）

4. 料理名の大きさ: 32pt、太字
5. 説明文: 14pt
6. 価格: 20pt、色 #dc143c（深紅）

### ステップ5: 定番メニューを追加（5分）
1. 「テキスト」→「本文を追加」
2. [CANVA_MENU_TEXT.txt] から「【 定番・家庭料理 】」以下をコピー＆ペースト
3. レイアウトを整える:
   - セクションタイトル: 24pt、太字
   - 料理名: 18pt、太字
   - 説明文: 12pt
   - 価格は右揃え

4. 「要素」→「線」で区切り線を追加

### ステップ6: その他のセクションを追加（5分）
1. 同様に「季節の料理」「お子様メニュー」「ドリンク・デザート」を追加
2. 各セクション間に15mm以上の余白を入れる

### ステップ7: フッターを追加（2分）
1. 「テキスト」→「本文を追加」
2. 以下を入力:

```
【 営業案内 】
営業時間: 11:00 〜 20:00
定休日: 水曜日

【 お問い合わせ 】
TEL: 0982-XX-XXXX
住所: 宮崎県東臼杵郡門川町

※メニューは季節や仕入れ状況により変更となる場合がございます
```

### ステップ8: 装飀を追加（任意）（3分）
以下の要素を検索して追加すると、より素敵になります：

- 「和風 フレーム」
- 「日本食 イラスト」
- 「料理 アイコン」
- 「区切り線 和風」

### ステップ9: 確認と調整（2分）
1. 全体のバランスを確認
2. フォントサイズを調整
3. 色のバランスを確認
4. 誤字脱字をチェック

### ステップ10: 保存＆公開（1分）
1. 右上の「共有」をクリック
2. 「その他」→「リンクをコピー」
3. URLからデザインIDをメモ:
   - URL: `https://www.canva.com/design/DAF_xxxxxxxxxx/Edit`
   - デザインID: `DAF_xxxxxxxxxx`

4. `.env` ファイルにデザインIDを入力:
   ```
   CANVA_MENU_DESIGN_ID=DAF_xxxxxxxxxx
   ```

## 🆕 トラブルシューティング

### Q: テキストが表示されない
A: フォントがサポートされていない可能性があります。「Noto Sans JP」や「Noto Serif JP」を使用してください。

### Q: レイアウトが崩れる
A: テキストボックスのサイズを調整してください。ドラッグしてサイズを変更できます。

### Q: 色が反映されない
A: カラーコードの前に `#` を付けてください（例: `#8b4513`）

### Q: デザインIDが見つからない
A: CanvaデザインのURLを確認してください。`design/` の後ろの文字列がデザインIDです。
