# 🍴 メニュー表PDF自動生成ガイド

旬食工房きよし - Pythonで美しいメニュー表のPDFを自動生成！

## ✨ 特徴

- ✅ **完全無料** - CanvaやAdobe Illustratorは不要
- ✅ **自動生成** - ワンクリックで完成
- ✅ **和風モダン** - 温かみのある配色
- ✅ **印刷対応** - A4縦サイズ、そのまま印刷可能

---

## 📋 前提条件

- Python 3.7以上がインストールされていること
- インターネット接続（ライブラリのダウンロード用）

---

## 🚀 セットアップ手順

### ステップ1: Pythonをインストール（まだの場合）

1. [Python公式サイト](https://www.python.org/downloads/) にアクセス
2. Python 3.10以上をダウンロード
3. インストール時、「Add Python to PATH」にチェックを入れる

### ステップ2: 必要なライブラリをインストール

ターミナル（またはコマンドプロンプト）を開いて、以下を実行：

```bash
cd kioshi-restaurant
pip install -r requirements-pdf.txt
```

または、個別にインストール：

```bash
pip install reportlab==4.0.7
pip install Pillow==10.2.0
```

### ステップ3: PDFを生成

```bash
python generate_menu_pdf.py
```

これだけ！`kioshi-restaurant-menu.pdf` が生成されます🎉

## 🔧 カスタマイズ方法

### メニュー内容を変更したい場合

`menu_data.py` を編集してください：

```python
# 例: 価格を変更
"price": 1500  # → "price": 1600

# 例: メニューを追加
{
    "name": "新しいメニュー",
    "description": "説明文",
    "price": 1000
}
```

### デザインを変更したい場合

`generate_menu_pdf.py` の色やフォントサイズを調整してください：

```python
# 色の変更
textColor=colors.HexColor('#8b4513')  # → お好きな色コードに

# フォントサイズの変更
fontSize=14  # → fontSize=16
```

## ❓ トラブルシューティング

### Q: 「ModuleNotFoundError: No module named 'reportlab'」と出る

A: ライブラリがインストールされていません。以下を実行：

```bash
pip install reportlab
```

### Q: 日本語が文字化けする

A: Windowsの日本語フォントが見つからない可能性があります。`generate_menu_pdf.py` のフォントパスを確認してください。

### Q: PDFが開けない

A: PDFビューア（Adobe Acrobat Readerなど）をインストールしてください。

## 🔄 メニュー更新の手順

メニューを変更する場合：

1. `menu_data.py` を編集
2. `python generate_menu_pdf.py` を実行
3. 新しいPDFが生成される
4. 完了！
